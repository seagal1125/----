系統化教學帶你入門快打旋風經典模式第一集 基本拳腳 #快打旋風6 #教學 #新手

系統化教學帶你入門快打旋風經典模式第一集：基本拳腳

(00:00) 前言與目標客群

  • 目標觀眾：從未接觸過格鬥遊戲的新手、卡在銅銀牌無法進步的玩家、或想從現代模式轉經典模式的玩家。
  • 教學理念：參考 UP 主「茶茶山」的循序漸進邏輯，帶領玩家系統化入門。

(01:46) 遊戲模式介紹

  • Fighting Ground (格鬥之地)：本系列教學的主要模式。
  • World Tour (環球遊歷)：RPG 模式，雖然好玩但對提升對戰實力幫助有限。
  • Battle Hub (對戰大堂)：可與全球玩家對戰，建議有基礎後再進入以免挫折。
  • 練習室 (Training Mode)：點選 Fighting Ground -> 練習 -> 練習室，這是變強最重要的地方。

(03:23) 角色與控制器選擇

  • 角色推薦：新手建議從 隆 (Ryu) 開始。
    • 原因：帥氣、擁有格鬥遊戲最基礎的工具，練好隆能輕鬆轉換到其他角色。
  • 操作模式：本教學專注於 經典模式 (Classic)。
  • 控制器：搖桿 (Arcade Stick)、Hitbox、鍵盤皆可。
    • 鍵盤設定建議：模擬 Hitbox 配置。建議將「跳躍」設為空白鍵 (拇指)，形成 ASD (左下右) + Space (跳) 的佈局，操作反應會更快。

(07:15) 訓練場基礎設置

  • 顯示設定：
    • 開啟「顯示攻擊資訊」。
    • 開啟「輸入歷史顯示」(左側可看按鍵指令)。
    • 開啟「虛擬控制器」。
  • 基本移動：前進 (6)、後退 (4)、蹲下 (2)、跳躍 (8/7/9)。
    • 數字鍵盤表示法 (Numpad Notation)：以鍵盤數字鍵對應方向，如 2 是下，6 是前，5 是回中 (不按)。

(08:54) 基本拳腳 (Normals)

  • 六鍵配置：輕拳 (LP)、中拳 (MP)、重拳 (HP)、輕腳 (LK)、中腳 (MK)、重腳 (HK)。
  • 18種基本攻擊：結合 站立 (Stand)、蹲下 (Crouch)、跳躍 (Jump) 三種狀態 x 6 顆按鍵。
    • 例如：5HP (站重拳)、2MK (下中腳)。
    • 5 代表站立回中 (什麼方向都不按)。

(10:42) 練習重點：距離感

  • 感受距離：在訓練場把防禦設為「不防禦」，練習各種拳腳的最遠打擊距離。
  • 關鍵拳腳：熟悉如 2MK (下中腳)、5HP (站重拳) 等長距離牽制技。
  • 跳躍攻擊：練習跳攻擊的打點，太早出招會打不到，太晚出會落地。

(12:43) 攻擊屬性與防禦機制

  • 上段攻擊 (High)：站防、蹲防皆可擋住 (大多數站立拳腳)。
  • 下段攻擊 (Low)：必須 蹲防 (如下中腳/下輕腳)。
  • 中段攻擊 (Overhead)：必須 站防 (如跳攻擊是典型的中段)。

(13:18) 防禦策略

  • 基本防禦姿勢：預設採用 蹲防 (拉住下後/1)。
    • 優點：同時防禦上段與下段。
  • 應對跳躍：看到對手跳起，立即改為 站防 (拉後/4) 以防禦空中攻擊 (中段)。

(14:51) 摔技 (Throws) 與解摔

  • 摔 (Throw)：當對手一直防禦時，用摔 (LP+LK) 破防。
  • 解摔 (Tech Throw)：被摔的瞬間同時按摔 (LP+LK) 拆開。
  • 練習設定：在防禦設定開啟「解摔練習」，成功會顯示「解摔」。

(16:30) 必學功能：重置戰鬥 (Reset)

  • 快捷鍵設定：在「快捷鍵操作設定」中綁定「重置戰鬥」。
  • 功能：一鍵回到場中初始位置，省去等待時間，大幅提升練習效率。

(19:02) 綜合練習與心態建立

  • 練習目標：
    • 設定電腦為 LV2 或 LV3，專注於「用最遠距離打到對方」。
    • 不管對手出什麼，專注執行自己的練習（例如抓下中腳距離）。
    • 2P 側 (面向左) 也要特別練習，適應反向操作。
  • 新手心態：
    • 不要想著一開始就學連段、學必殺技。
    • 在練習場不用待太久，大概抓到感覺即可進入實戰。
    • 目標不是贏：設定小目標（例如：這場我只要從最遠距離踢到他），在實戰中確認自己學會了什麼。

(24:49) 排位賽實戰驗證

  • 為什麼打排位賽 (Ranked Match)：
    • 排位賽最容易遇到實力相當的對手，體驗比休閒賽更好。
    • 即使遇到炸魚或外掛，也是幾分鐘的事，封鎖即可。
  • 網路設定：建議設定 3-5 格或僅限 5 格，強烈建議使用 有線網路 以確保體驗。
  • 實戰策略：
    1. 地面戰：不管是防禦還是進攻，專注用長距離拳腳 (2MK, 5HP) 牽制。
    2. 對空：看到對手跳起來，記得切換為站防。
    3. 被壓制時：被逼到角落就一直拉著「前跳」逃離，或者專注防禦找機會。
    4. 無視勝負：輸了沒關係，重點是練習「下中腳摸人」、「跳攻擊抓距離」。
  • 常見狀況應對：
    • 對手一直 DI (衝擊)？ -> 沒關係，新手先不管，繼續做自己的練習。
    • 對手一直摔？ -> 沒關係，多跳幾次試著逃開，或者專注於自己的攻擊距離。

(51:46) 第一集總結

  • 核心重點：熟悉基本拳腳距離，學會基本的「蹲防擋下段、站防擋跳躍」。
  • 下集預告：將會講解必殺技的使用。

系統化教學帶你入門快打旋風經典模式之第二集__指令表_轉絕招__快打旋風6__教學__新手

系統化教學帶你入門快打旋風經典模式之第二集：指令表與轉絕招

(00:00) 進入訓練室與指令表

  • 進入方式：在標題畫面按 X 直接進入訓練室。
  • 方向術語：前 (6)、後 (4)、下 (2)、上 (8)。以角色面向為基準，「前」永遠是面向對手的方向。

(00:24) 必殺技輸入 (Command Inputs)

  • 數字鍵盤表示法：
    • 236P (正轉手)：下(2) -> 前斜下(3) -> 前(6) + 拳(P)。即「波動拳」。
    • 214K (反轉腳)：下(2) -> 後斜下(1) -> 後(4) + 腳(K)。即「龍捲旋風腳」。
    • 236K (正轉腳)：上段足刀。
    • 214P (反轉手)：波掌擊。

(03:01) 昇龍拳指令 (623P) 與簡化按法

  • 標準指令：前(6) -> 下(2) -> 前斜下(3) + 拳。
  • Hitbox/鍵盤 常見簡化戳法：
    1. 626：前(按住) -> 下(點一下) -> 前(放開下) + 拳。
    2. 266：(理論上是323? 影片中提到的一種變體，需視控制器而定)。
    3. 636：按住前 -> 中指滑過下 -> 拳。
    4. 628 (作弊法)：按住前(6) + 下(2) -> 同時按上(8) + 拳。(利用 SOCD 覆蓋機制)。
  • 建議：找到最適合自己手感的按法即可，不必拘泥於標準 623。

(04:58) OD 必殺技 (Overdrive)

  • 發動方式：輸入指令時同時按兩個拳或兩個腳 (例如 236PP)。
  • 特性：消耗兩格動力槽 (Drive Gauge)，招式性能強化、有裝甲或無敵時間等。

(05:46) 超必殺技 (Super Arts / SA)

  • SA1：236236 + 腳 (視角色而定，龍可能是 K 或 P，影片口誤主要強調兩次正轉/反轉)。
    • 修正註記：龍的 SA1 真空波動拳通常是 236236P。
  • SA2：214214 + P (龍的真波掌擊)。
  • SA3 (CA)：236236 + P (真昇龍拳)。血量低於 25% 變為 CA (Critical Art)，傷害提升。
  • 手勢技巧：反轉 (214214) 對新手較難，建議手掌帶動手指去滑動。

(07:11) 其他指令機制

  • 蓄力/集氣 (Charge)：如春麗、古烈。需按住方向 (如下或後) 一段時間，再推反方向+攻擊。
    • 4蓄6 (集氣後前)、2蓄8 (集氣下上)。
  • 指令投 (Command Throw)：如桑吉爾夫 (Zangief)。
    • 360度：轉一圈。Hitbox/鍵盤只需按出 上下左右四個方向 (如 6248) 即可觸發。
    • 應對：指令投無法拆投 (解摔)，只能 跳躍 躲避。

(08:03) 常見新手問題 (Troubleshooting)

  • 出招失敗：
    • 原因通常是按太快、不乾淨。
    • 解決：放慢速度，確保指令輸入準確 (Clean Inputs)。觀察虛擬控制器的輸入歷史。
  • 前走波動變昇龍 (Walk Forward Hadoken -> DP)：
    • 原因：前走 (6) 殘留，接波動 (236) 會被系統判定為 6236 (昇龍)。
    • 解法 1：輸入 236 前多按一個後 (4)，變成 6...41236 (類似半圈)。
    • 解法 2 (推薦)：養成 回中 (Neutral) 習慣。前走停頓一下 (手放開) 再輸入 236。
  • 連續波動出不來：畫面中同時只能存在一顆波動拳。

(17:16) 實戰練習目標

  • 在實戰中穩定搓出波動拳 (輕/中/重/OD)。
  • 嘗試加入簡單連段 (如重拳接 OD 波動)。
  • 不要在意輸贏，重點是指令成功率。

系統化教學帶你入門快打旋風經典模式之第三集：取消與復合

(00:00) 什麼是取消 (Cancels)

  • 定義：在普通拳腳的「後搖」(Recovery / 藍色影顯示) 結束前，輸入必殺技指令，直接接續出招。
  • 設定：開啟「顯示取消時機」。
    • 角色變 紅色：表示該時刻可以被取消。
    • 只有特定拳腳可以被取消 (如龍的 2MK、5MP)。

(01:18) 連段成立的三要素

  1. 拳腳可被取消 (Cancellable)：如戰中拳 (5MP)、下中腳 (2MK)。(5MK 通常不可取消)。
  2. 必殺技發生速度夠快 (Startup)：如果接太慢的招會連不上。
  3. 距離足夠：太遠可能打不到。

(02:17) 基礎取消練習

  • 練習目標：
    • 5MP (站中拳) xx 236HP (重波動拳)
    • 2MK (下中腳) xx 236HP (重波動拳)
  • 手感：不要等拳腳打完才轉，要在拳腳打中對手的瞬間輸入完必殺技指令。
  • 狂按法 (Mash)：新手若覺得節奏難抓，可以在出拳後瘋狂轉指令和按鍵，確保招式出來 (雖然不優雅但有效)。

(04:51) 取消接 OD 與 SA

  • 必殺技不僅能取消普通攻擊，OD 必殺技 和 SA (超必殺) 也可以取消普通攻擊。
  • 練習：5HP xx OD 波動、2MK xx SA3 (236236P)。

(09:10) 復合指令 (Buffering / Option Select 概念)

  • 概念：在普通攻擊 (如 Poke / 牽制) 的同時，預先輸入必殺技指令。
    • 原理：
      • 如果普通攻擊 打中 (Hit) 或 被擋 (Block) -> 產生動作停頓 (Hitstop) -> 系統讀取到必殺技指令 -> 招式取消出來。
      • 如果普通攻擊 揮空 (Whiff) -> 沒有 Hitstop -> 拳腳動作極快結束 -> 系統還沒讀到完整指令或錯過窗口 -> 必殺技不出來。
  • 應用：2MK (下中腳) + 波動拳指令。
    • 在中距離用 2MK 摸對手，同時刷波動拳。
    • 打到人 -> 自動出波動拳 (連段)。
    • 沒打到 -> 只有 2MK (安全)。
  • 回中 (Neutral) 的重要性：復合時要避免一直拉著前或後，最好在動作與動作間有回中，以免誤出昇龍或其它招式。

(11:46) 實戰心態與練習

  • 實戰目標：嘗試在對戰中使出「2MK xx 波動拳」。
  • 面對現代模式：現代模式反應快 (一鍵昇龍/SA)，但只要專注練習自己的目標 (取消連段)。

(14:30) 實戰應用：無腦復合

  • 策略：在立回中，只要按下 2MK，手就自動跟著轉波動拳的指令。
    • 完全依賴系統判斷：打中就出波，沒打中就不出。
    • 優勢：不需要反應時間確認是否打中，降低立回時的心理負擔。

(22:00) 實戰案例分析

  • 面對一直跳的對手：
    • 如果對手喜歡亂跳，復合波動可能會有些風險 (被跳入確反)。
    • 但在新手階段，先練好 "只要出腳就復合" 的肌肉記憶更重要。
  • 被摔怎麼辦：
    • 新手階段不會解摔 (Throw Tech) 很正常。
    • 寧願被摔，也要專注在練習 取消 和 復合 這兩個核心技巧。
    • 不要因為怕被摔而打亂自己的進攻節奏。

(32:15) 總結

  • 取消 (Cancel) 是格鬥遊戲最基礎也最重要的進攻手段。
  • 復合 (Buffer) 則是立回神器，能讓你不知不覺中打出高手般的確認連段。

系統化教學帶你入門快打旋風經典模式第四集：壓起身與目押

(00:00) 前言

  • 入門定義：作者認為達到「白金一 (Platinum 1)」才算真正入門，此前只要連勝就能快速升分。

(00:44) 壓起身 (Okizeme / Meaty)

  • 原理：
    • 對手倒地起身時有短暫無敵時間 (黃色)。
    • Meaty (壓起身)：讓攻擊的「持續幀 (Active Frames / 紅色)」剛好覆蓋在對手無敵結束的瞬間。
    • 成功：對手亂動 (Mash) 會被判定為康特 (Counter Hit)。
    • 太慢：變成攻擊的前搖 (Startup / 綠色) 撞到對手的攻擊 -> 被打。
    • 太快：攻擊揮空 (打在無敵上) -> 收招 (Recovery / 藍色) 被對手確反 (Punish Counter)。
  • 設定練習：設定假人「倒地反攻」為「輕拳 (4F)」。目標是在他起身時用中拳打卻他 (打出 Counter)。

(09:05) 連段原理：目押 (Link)

  • 目押：在前一個攻擊動作完全結束後，利用幀數優勢 (Frame Advantage)，在特定時機輸入下一個攻擊。
  • 與取消的區別：取消是截斷收招；目押是等收招結束。
  • 練習：5MP (站中拳) -> 5MP。
    • 需掌握節奏，不能狂按。
    • 距離限制：如果太遠，第二下會因推不開 (Pushback) 而打不到，通常需要貼身。

(10:46) 綜合連段練習 (Divide and Conquer)

  • 拆解練習法：
    1. 練熟 5MP -> 5MP (目押)。
    2. 練熟 5MP -> 236P (取消)。
    3. 組合：5MP -> 5MP -> 236P。
  • 倒地後壓制：5MP -> 5MP -> 236P -> (對手倒地) -> 走過去 -> 5MP (壓起身)。

(12:26) 打摔二擇 (Strike / Throw Mixup)

  • 當對手學會防禦你的壓起身時 (不敢亂動了)：
    • 打 (Strike)：壓制對手亂動或拆摔。
    • 摔 (Throw)：對付死防 (Blocking) 的對手。
  • 心理戰循環：
    1. 你壓起身 (打) 勝過 對手亂動。
    2. 你摔 (Throw) 勝過 對手防禦。
    3. 對手拆摔 (Tech) 平手 或 勝過 你的摔。
    4. 你後退打拆摔 (Shimmy) 勝過 對手拆摔 (進階概念)。

(17:50) 實戰資源管理與應用

  • 實戰心態：在訓練場不用練到 100%，大概懂邏輯就去實戰。實戰中把「壓起身」和「打摔擇」作為目標。
  • 73 幀倒地：普通倒地起身通常是 58 幀，但某些招式 (如掃腿打康特、SA 打中) 會造成 73 幀或更長的倒地時間。
    • 這會改變壓起身的時機，需要經驗累積。

(20:53) 實戰分析：對付亂動的對手

  • 現象：低分段 (如銀牌) 玩家非常喜歡在起身時亂按 (Mash)。
  • 對策：嚴格執行壓起身 (Meaty)。
    • 只要你壓得準，對手亂按必定被 Counter。
    • 可以用簡單的 5MP -> 5MP 確認連段，打掉對手一大管血。
    • 不需要複雜的擇，只要「壓起身」做對，就能打崩亂動的對手。

(29:48) 實戰分析：對付防守好的對手

  • 現象：對手開始懂得起身防禦，不再亂按。
  • 對策：摔 (Throw)。
    • 壓起身時直接走過去摔。
    • 如果對手開始解摔 (Tech)，則回到 打 (Strike)，或者用 Shimmy (後退一點點再打) 來騙他解摔揮空。

(36:50) 總結

  • 壓起身 (Okizeme) 是擴大優勢的關鍵。
  • 透過「打」與「摔」的切換，給對手巨大的心理壓力。
  • 練好壓起身，就算連段不強，光靠抓起身也能在白金以下輕鬆取勝。

系統化教學帶你入門快打旋風經典模式第五集_凹_大確反__快打旋風6__教學__新手

系統化教學帶你入門快打旋風經典模式第五集：凹與大確反

(00:00) 版本機制解說 (後受身)

  • 改動：現在普通倒地也可以「後受身 (Back Roll)」。
  • 影響：這讓進攻方在場中很難持續保持貼身壓制。這設計是為了避免一瞬間的滾雪球效應。

(03:41) 什麼是「凹」(Reversal / Ou)

  • 定義：在自己處於劣勢 (如倒地起身、被壓制) 時，賭博性地使用具有「無敵屬性」的招式。
  • 常見招式：
    • OD 昇龍拳 (OD DP)：全身無敵。
    • SA1 / SA3：通常具有無敵時間 (SA1 對飛行道具不一定無敵，視角色而定)。
  • 風險：無敵技被防住後會有極大的硬直 (Recovery)，必定被大確反 (Punish Counter)。

(07:11) 凹的時機

  1. 起身凹：賭對手會壓起身 (Meaty)。
  2. 連段空隙凹：對手連段失誤或壓制有縫隙 (非連續防禦) 時，狂按昇龍插進去。
  3. 反應凹：看到對手綠衝 (Drive Rush) 或跳入，直接 SA/昇龍。
  4. 負幀凹 (Gambling)：雖然自己是不利幀數 (如被擋 -2)，但賭對手會出手，強行出無敵技。

(10:21) 應對與確反 (Punish Counter)

  • 大確反練習：
    • 設定電腦倒地後出 OD 昇龍。
    • 你壓起身 -> 防禦 (擋住昇龍) -> 等他落地。
    • 反擊：這時對手處於嚴重硬直，使用 5HP (站重拳) 起手 的最大傷害連段。
    • 連段範例：5HP (PC) -> 5HP -> 236HP / SA3。
  • 進階：如果是對手 SA3 被擋，硬直更長，可以從容地用最強連段 (如綠衝起始) 去修理他。

(12:30) 攻防猜拳總結 (RPS)

  • 打 (Strike) 贏 亂動/拆摔。
  • 防 (Block) 贏 凹 (Reversal)。
  • 摔 (Throw) 贏 防 (Block)。
  • 凹 (Reversal) 贏 打/摔 (因為無敵)。

(15:00) 實戰應用：不要怕凹/也不要怕被凹

  • 新手迷思：新手常因為怕對手凹而不敢壓起身。
    • 糾正：必須要壓！如果不壓，對手起身就沒有壓力。
    • 偶爾被凹到沒關係，重點是你要逼對手猜拳。如果他凹失敗一次，你的一套大確反傷害通常比他凹中你的傷害高得多 (期望值問題)。
  • 新手迷思 2：自己不敢凹。
    • 糾正：要在實戰中練習凹。這是一種強力的防守反擊手段，能讓對手在進攻時有所忌憚。

(20:48) 實戰分析：對付凹魔人 (Wake-up DP Spammers)

  • 現象：有些對手 (如 Ken/Luke) 只要倒地就喜歡凹昇龍。
  • 對策：防禦 (Block)。
    • 壓起身時，假裝要打，其實是走過去防禦 (Shimmy 的變體或直接蹲防)。
    • 一旦擋住他的無敵技 -> 大確反一套帶走。
    • 觀察對手資源：如果對手沒氣 (Burnout)，他就凹不出 OD 昇龍 (只能凹 SA)，這時可以更放心壓制。

(28:00) 總結

  • 凹 (Reversal) 是格鬥遊戲心理戰的高潮。
  • 學會「大確反」是提升勝率的關鍵。擋住昇龍後只摔他是虧的，一定要練一套痛的連段。
  • 實戰中保持平常心，被凹了就被凹了，不要生氣，下一把猜回來就好。

系統化教學帶你入門快打旋風經典模式第六集：確認與確反

(05:28) 攻擊確認 (Hit Confirm)

  • 概念：在攻擊打中對手時 (Hit) 繼續連段，被防住時 (Block) 停止或接安全招式。
  • 練習：
    • 2MP (下中拳) / 5MP (站中拳) 連打兩下。
    • 打中 -> 取消接必殺技 (如波動拳、OD 足刀、龍卷)。
    • 被擋 -> 停手，或者波掌擊 (214P，較安全)。
  • 單點確認 vs 多點確認：新手建議用兩下拳腳來確認 (例如 5MP > 2MP)，反應時間較充裕。單發確認 (Single Hit Confirm) 對反應要求極高，通常是高手領域。

(07:07) 確反 (Punish / Punish Counter)

  • 原理：
    • 當對手招式打在防禦上，會產生硬直差 (Frame Advantage)。
    • 如果對手硬直差 (例如 -6) 大於你最快招式的發生時間 (例如 4F)，你就能 必中 反擊。
  • 條件：
    1. 幀數 (Frames)：對手夠負 (通常 -4 以上)。
    2. 距離 (Spacing)：你的招式打得到。有些招式雖然很負 (如 -10)，但如果距離太遠，輕拳摸不到，就無法確反。

(08:24) 重點練習：確反掃腿 (下重腳 / 2HK)

  • 通用規則：
    • 幾乎所有角色的下重腳 (Sweep) 都是 大確反 (通常 -10 到 -12 以上)。
    • 龍的下重腳發生約 9F，所以 「擋住下重腳 -> 確反下重腳」 是基本功。
  • 距離陷阱：有時候對手用腳尖 (Tip / 尖端) 摸你，雖然幀數很負，但你的下重腳可能會揮空。這時需要用較長的招應對。

(12:30) 關於硬直差的知識

  • 不需要背表：新手不需要背誦所有角色的幀數表。
  • 記動畫：記住每個角色「下重腳」的動作，看到擋住 -> 肌肉記憶按重腳。
  • 安全招與騙招：
    • 波動拳貼身被擋是不安全的 (-6 左右)，可能被確反。
    • 但拉遠距離後，因為飛行時間吃掉了幀數，波動拳可能變安全甚至有利 (+)。

(15:32) 安全距離與幀數變化

  • 許多招式 (如足刀 236K) 在貼身被擋是確反，但在遠距離 (Tip range) 被擋可能變得很安全 (-2 或 -3)。
  • 實戰中要學會分辨距離，不要看到招式就無腦確反，可能會被打。

(20:00) 實戰應用與觀察

  • 觀察對手：
    • 如果對手喜歡亂出下重腳 (Sweep)，專注擋住並掃回去。
    • 如果對手喜歡出不安全的必殺技 (如貼身波動、貼身足刀)，嘗試用輕攻擊確反。
  • Match Analysis (實戰分析)：
    • 影片中演示了對戰 Zangief (老桑)。老桑的某些招式距離遠很難確反，這時不要強求。
    • 面對 Modern controls (現代模式) 的 Ed，要注意他的快節奏。
    • 即便到了金牌段位，很多玩家依然會習慣性地出下重腳，這就是你的機會。

(31:00) 總結

  • 確認 (Hit Confirm) 讓你進攻更安全，不會因為亂出必殺技被確反。
  • 確反 (Punish) 是免費的傷害。擋住下重腳 = 免費 KD (擊倒) + 壓起身機會。
  • 練好這兩點，你的防守反擊能力會大幅提升。

系統化教學帶你入門快打旋風經典模式第七集_對空_波升__快打旋風6__教學__新手

系統化教學帶你入門快打旋風經典模式第七集：對空與波升

(00:00) 對空 (Anti-Air) 的重要性

  • 收益：跳入攻擊收益極高 (一套連段 3000+ 傷害)。如果不對空，對手會肆無忌憚地跳。
  • 目標：讓對手不敢跳，強迫他們在地面跟你立回。

(02:44) 對空的三種方式

  1. 昇龍拳 (Shoryuken / DP)：
    • 最強對空：某些版本 (如中昇龍、OD昇龍) 擁有「對空無敵」屬性。
    • 只要指令對、時機對，幾乎必勝。
  2. 普通拳腳 (Normals)：
    • 如龍的 2HP (下重拳)。
    • 優點：簡單、只需按一個鍵。
    • 缺點：沒有無敵，可能相殺或被打掉，且需注意距離。
  3. 空對空 (Air-to-Air)：
    • 跳起來按拳/腳 (如跳中拳、跳輕拳)。
    • 適用於反應慢了、距離不適合昇龍，或想爭取主動權時。

(07:11) 波昇戰術 (Fireball Trap / "Bo-Sheng")

  • 核心邏輯：
    1. 發波動拳 (Hadoken) 騷擾對手。
    2. 對手覺得煩/不想防禦損血 -> 跳起來 想攻擊你。
    3. 你預判/反應到跳躍 -> 昇龍拳對空。
  • 安全波動：
    • 只有在對手「跳入距離」之外發波才安全。
    • 如果太近發波，對手跳入會剛好踢到你的收招僵直 (Punish)。
  • 練習：
    • 錄製電腦：防禦後「跳攻擊」。
    • 練習：發波 -> 看電腦跳 -> 昇龍。

(12:00) 相信你的昇龍

  • 昇龍拳在起手幀有對空無敵。
  • 即使看起來對手的腳已經踢到你的頭，只要昇龍出來，系統判定你會贏。
  • 心態：不要怕被打，勇敢出昇龍。

(17:54) 波對波 (Fireball War)

  • 判定優先級：
    • SA (超必殺) 波 > OD (強化) 波 > 普通波。
    • Hit 數：2 Hit 的波會吃掉 1 Hit 的波，然後剩下 1 Hit 繼續飛。
  • 電刃鍊氣 (Denis Charge)：強化波動拳 (2 Hits / 速度快)，能贏過普通波。
  • 不知火舞 (Mai) 案例：她的 SA1 可以強化扇子，要注意其 Hit 數變化。

(20:00) 實戰應用：波昇的距離感

  • 黃金距離：剛好是波動拳能打到對手，但對手跳入會落在你昇龍範圍內的距離。
  • 對付穿波招式：
    • 對付 Vega (Bison) 的穿波 (Psycho Crusher)，要保持距離或垂直對應。
    • 如果對手有穿波技，發波頻率要降低，或者用輕波騙招。

(29:30) 實戰分析：對戰 DJ

  • DJ 的雙波 (Double Air Slasher) 很強，不要盲目對波。
  • 保持距離，用 Parry 接波回氣。
  • 當他忍不住跳過來時，給予昇龍痛擊。
  • 此階段 (白金以下)，只要對手跳不過來又對不過波，基本上就會自亂陣腳。

(36:00) 總結

  • 對空需要分配注意力 (Mental Stack)。
  • 初期做不到「自動對空」很正常，先強迫自己意識到「他可能會跳」。
  • 你的昇龍是你最強的盾，也是最強的誘餌。

系統化教學帶你入門快打旋風六經典模式第八集_幀數_下中腳取消綠衝

系統化教學帶你入門快打旋風六經典模式第八集：幀數 下中腳取消綠衝

(00:00) 幀數觀念 (Frame Data)

  • 不利 (Negative / Minus)：
    • 當你的攻擊被防住後，若硬直差是負數 (如 -1, -2, -3)，代表對手可以比你先動。
    • 重點：處於不利幀數時，絕對不要亂按 (Don't mash)。尤其是對手防禦後，如果你還想點輕拳 (4F)，會被對手的輕拳 (4F) 搶招打成 Counter。
  • 有利 (Positive / Plus)：
    • 若硬直差是正數 (如 +2, +3)，代表你可以比對手先動。這時對手如果亂動就會被打。

(02:53) 獲取有利幀的方法：藍防 (Drive Parry)

  • 影片中詳細介紹了 藍防 (Parry) 的操作與優勢。
    • 操作：中拳+中腳 (MP+MK)。
    • 特點：對打擊和飛行道具無敵 (除了摔)。
    • 完美藍防 (Perfect Parry)：在被攻擊到的瞬間按 Parry，幀數優勢極大，可以反擊 (但傷害修正 50%)。
    • 普通藍防：不會被退氣，且成功防禦會回氣。對於不想被對手連續壓制削減動力槽 (Drive Gauge) 時非常有用。

(07:07) 下中腳複合綠衝 (2MK DRC Option Select)

  • 輸入技巧：2MK (按住不放) + 66 (有的 Hitbox 技巧) 或 2MK xx Parry 鍵。
  • 目的：在立回時，每次出 2MK 都預輸入綠衝。
    • 如果 2MK 沒打到 (揮空) -> 綠衝不會出來 (省氣)。
    • 如果 2MK 打到 (Hit/Block) -> 綠衝自動出來 -> 形成壓制 (+4) 或連段。
  • 連段選擇：
    • 打中：2MK > DRC > 5MP > 2HP > ... (痛連段)。
    • 被擋：2MK > DRC > 5MP (有利) > 2MP ... (繼續壓制)。

(12:00) 前重拳 (6HP) 的強大

  • 這是一個 正幀 (+) 的普通技。
  • 新手不知道這個概念，看到你打完會想動，結果被你下一招打 Counter。
  • 應用：6HP (被擋) -> 5MP (站中拳)。
    • 無腦壓制，對方亂動就會被打。
    • 距離注意：如果在較遠距離打中 6HP，可能 5MP 接不到，要改接 2MP。

(16:39) 實戰資源管理 (Drive Gauge Management)

  • 不要無腦綠衝：因為綠衝取消很貴 (3格)。
  • 動力槽策略：
    • > 5 格 (健康)：可以嘗試用 2MK 複合綠衝進攻。
    • < 3 格 (危險)：停止綠衝，轉為保守，發波、對空、立回，等待氣回復。被 Burnout (白血) 是非常劣勢的狀態。
    • 進攻節奏：如果你綠衝進去打了幾下沒破防，不要硬打，退回來回氣，保持資源優勢。

(20:46) 實戰分析

  • 利用正幀欺負新手：很多對手不知道 6HP 是正的，會一直點拳。你就一直出 6HP > 5MP 打爆他。
  • 2MK 複合的穩定性：這招是龍 (Ryu) 為數不多能主動起手的強勢手段，必須練熟。
  • 面對 Burnout：如果你把對手打到 Burnout (氣絕)，可以盡情壓制，甚至用 Impact (DI) 把他撞牆暈眩。

(26:00) 總結

  • 幀數是格鬥遊戲的數學，懂數學的人可以「合法作弊」(利用正幀)。
  • 複合綠衝是現代格鬥遊戲最重要的立回技巧之一，能讓你的反應看起來像職業選手。

系統化教學帶你入門快打旋風經典模式_第九集_DI_動力衝擊

系統化教學帶你入門快打旋風經典模式第九集：DI (動力衝擊)

(00:00) DI 基本屬性 (Drive Impact)

  • 指令：重拳 + 重腳 (HP + HK)。消耗 1 格動力槽。
  • 霸體 (Armor)：
    • 起手階段有霸體，可以承受 2 次 攻擊。
    • 弱點：第 3 次攻擊會打破霸體 (Armor Break)。破甲屬性招式 (如多段判定的波、老桑蓄力拳) 也能破。
  • 發生與硬直：發生 26F (慢)，但後搖 (Recovery) 很長，若揮空極易被確反。

(02:25) DI 的 10 種交互狀況

  1. 被防住 (Block)：退 0.5 格氣，對手被推開 (-3F)。在版邊會被撞牆 (Wall Splat)。
  2. 打中 (Hit)：對手倒地，失 1 格氣。
  3. 打康特 (Counter)：對手硬直更長，失更多氣。
  4. 空中命中：對手被擊飛，可連段。
  5. 打收招 (Punish)：對手膝蓋跪地 (Crumple)，可接最大傷害連段。
  6. 打在藍防 (Parry) 上：對手回 2 格氣，且即使在版邊也 不會 撞牆。
  7. 打在金防 (Perfect Parry) 上：被大確反 (-35F)，但傷害修正 50%。
  8. 打氣絕 (Burnout) 對手：對手無法藍防，被撞牆會暈眩 (Stun)。
  9. 互撞 (Clash)：極少見，同時發生。
  10. 來得及取消但沒按：系統判定你可以反 DI 但你沒按，會有特殊特效。

(14:30) DI 的進階策略

  • 白血致死 (Chip Kill via DI)：
    • 雖然 DI 在版中被擋不會扣血 (白血會回)，但如果對手血量極低 (<400)，擋住 DI 也會因白血機制而死。
  • 反應時機 (Reaction)：
    • 透過綠衝 (DR) 或拳腳接觸時發動 DI，因為遊戲的 Hitstop (停頓感) 機制，會讓對手覺得時間變慢，反而更容易按回 DI。
    • 暗轉 (Screen Freeze) 技巧：在自己虛弱 (Burnout) 時，瘋狂轉 SA 指令但不按鍵。看到對手 DI 畫面變紅瞬間按下 SA 鍵，這叫「暗轉確認」。

(25:00) 什麼時候該用 DI？

  1. 瞎按 (Prediction/Gambling)：
    • 賭對手會出 不可取消 的重攻擊 (如龍的 2HK, 5MK)。這時他按不出反 DI，必中。
  2. 看確反 (Reaction)：
    • 看到對手出破綻大的招 (如龍的驢踢起手)，確認後按 DI。

(28:20) 什麼時候 不該 用 DI？

  1. 起身 DI：極差的選擇。因為對手壓起身通常用輕攻擊 (收招快)，或者直接投，都能贏 DI。
  2. 對空 DI：極差。對手跳攻擊會破甲或落地防禦。
  3. 連段中亂插 DI：沒有意義，增加被反風險，除非為了回氣或特殊連段。
  4. 場中無腦 DI：高分段玩家反應很快，場中隨意 DI 等於送對手一套連段。

(36:45) 總結

  • DI 是新手的殺手鐧，但隨著段位提升，使用頻率應降低。
  • 專注練習 反 DI (Counter DI)：聽到音效、看到紅色特效，肌肉記憶直接按回去。這比自己主動按更重要。

系統化教學帶你入門快打旋風經典模式_第十集_連續壓起身_角落壓制

系統化教學帶你入門快打旋風經典模式第十集：連續壓起身與角落壓制

(00:00) 為什麼要推版 (Corner Carry)

  • 版邊優勢：
    1. 版邊限定連段：許多高傷害連段只有在牆邊才能成立 (如 Loop)。
    2. 無需資源的壓起身：場中打倒對手常需要綠衝才能貼身，板邊則自動貼身。
    3. 限制對手移動：對手無法後退，你的招式更難揮空。

(01:43) 受身機制 (Recovery)

  • 種類：
    • 原地受身 (Normal Recovery)：倒地後直接原地起來。
    • 後受身 (Back Recovery)：倒地後往後滾一段距離再起來。
  • 操作：倒地時按任意兩鍵 (通常建議按著不用狂點)。
  • 為什麼要後受身：
    • 如果你原地受身，對手很容易就能繼續壓你。
    • 後受身能拉開距離，增加對手壓制的難度 (可能需要綠衝才壓得到)。
    • 例外：被強制倒地 (Hard Knockdown) 的招式 (如確反掃腿、確反投) 打中後，無法後受身。

(10:56) 壓持續 (Active Frame Meaty / "偷幀")

  • 概念：
    • 讓攻擊判定的「後半段」打中對手。
    • 這樣攻擊的「持續幀」結束得快，你的硬直差 (Frame Advantage) 會變大。
    • 例如：原本 +1 的招式，如果能用後半段打中，可能變成 +4。
  • 應用：
    • 龍的重昇龍 (HP DP) 結尾：這通常是固定 +37F (地面命中的話)。
    • 公式：昇龍 (+37) -> 空揮 5MK (消費幀數) -> 5MP (壓起身)。
    • 這樣壓到的 5MP 會變成 +2 (原本 +1) 甚至更多。

(20:13) 正一負一沒用？

  • 此作者的觀點：+1/-1 在實戰中強度有限，因為投的發生是 5F，輕拳是 4F。
  • 目標：透過「壓持續」製造 +2 或以上的優勢。
  • +2 的優勢：你的 4F 輕拳會贏過對手的 4F 輕拳 (相殺變單方面贏)，且能打贏對手的投。

(22:00) 角落壓制套路 (Corner Flowchart)

  • 龍卷旋風腳 (Tatsumaki) 結尾：
    • 輕龍卷 (+2) -> 輕拳 / 投 二擇。
    • 中龍卷 -> 傷害更高，壓制類似。
  • 重昇龍結尾 (版邊)：
    • 前衝 (Dash) x2 -> +3 左右，直接投或打。
    • 這個套路簡單暴力，適合新手。

系統化教學帶你入門快打旋風經典模式_第十一集_新手建議

系統化教學帶你入門快打旋風經典模式第十一集：新手建議

(01:06) 回合制遊戲觀念

  • 把格鬥遊戲當作超快節奏的回合制遊戲。
  • 我的回合 (My Turn / +Frames)：
    • 當你是正幀 (+)。
    • 選擇：進攻 (打、投)。
  • 對手的回合 (Their Turn / -Frames)：
    • 當你是負幀 (-)。
    • 選擇：防禦 (Block)。不要亂動！
  • 新手通病：在負幀時 (如被連段打完、升龍被擋) 不願意防禦，一直想按輕拳搶回來，結果一直挨打。

(05:29) 什麼時候是「負幀」？

  1. 攻擊被擋：大部分拳腳被擋都是負的 (除了少數如綠衝攻擊、龍的 6HP)。
  2. 被攻擊命中：絕大多數情況下，被打中後你都是負的。
    • 新手常覺得「被打了很不爽要打回去」，但這時按鍵只會被 Counter。
  3. 綠衝過來的攻擊：對手綠衝攻擊被你擋住，通常對手是正的 (+4)。這時要乖乖擋，不要動。
  4. 進入 Burnout (白血)：被擋的硬直增加 +4，變成極度不利。

(14:33) 觀察對手 (Adaptation)

  • 出題與解題：
    • 你出下中腳 (題目)，對手擋住後確反 (解題正確)。-> 你下次不能亂出。
    • 你跳入 (題目)，對手沒對空 (解題失敗)。-> 你可以一直跳，直到他學會為止。
  • 不要當單機玩家：不要不管對手在幹嘛，只顧自己打自己的套路。
  • 實戰應用：試探對手會不會對空、會不會解摔、會不會防凹。針對他的弱點一直打。

(20:06) 放慢節奏

  • 新手常處於「極度緊張/興奮」狀態，一開場就瘋狂進攻。
  • 建議：試著「蹲防」一陣子。
    • 觀察對手想幹嘛。
    • 等對手跳過來 -> 對空。
    • 等對手出大招 -> 確反。
    • 只要你能穩住防守，對手往往會自己送上門。

(21:56) 當個聰明的凹鬼 (Reversal Demon)

  • 不要每次都凹：容易被識破。
  • 延遲凹 (Delay Reversal)：擋住第一下攻擊後，在第二下凹。很多新手不懂得防連續凹。
  • 防凹：因為新手很愛凹，你在壓制時也要記得「防凹」。壓一下 -> 蹲防觀察 -> 確反。

系統化教學帶你入門快打旋風經典模式_第十二集_中段攻擊

系統化教學帶你入門快打旋風經典模式第十二集：中段攻擊

(00:57) 中段攻擊 (Overhead)

  • 定義：必須 站立防禦 (Stand Block) 的攻擊。蹲防會被打中。
  • 常見中段：
    • 所有角色的 跳躍攻擊 都是中段 (例外：Dhalsim 跳中手)。
    • 地面特殊技 (如龍的 6MP / 前中拳)。

(04:53) 中段的連段

  • 龍的前中拳 (6MP) 發生 20F，被擋 -1，打中 +3。
  • 綠衝中段 (Drive Rush Overhead)：
    • 綠衝 6MP -> 打中變成 +7，可以接 5MP -> 連段。
    • 這是一個強力的破防手段 (High/Low Mixup)。
    • 連段：DR 6MP -> 5MP -> 2MK -> 623HP (昇龍)。

(12:00) 延遲中段 (Fuzzy Guard / Delayed Overhead)

  • (進階概念) 高手會利用「延遲一點點出中段」來騙過對手的防守節奏。
  • 但對付新手通常不需要這麼複雜，直接出就能打中，因為新手往往只會蹲防。

(16:09) 殘局應用

  • 當對手血量很少時，心理壓力大，通常會下意識蹲防 (怕下段)。
  • 這時一個 綠衝中段 或 前中拳 往往能直接收割比賽 (偷中段)。

(17:40) 總結

  • 攻擊手段不能只有下段和投。
  • 加入中段攻擊 (配合綠衝)，能讓你的進攻變得立體，讓對手防不勝防。
